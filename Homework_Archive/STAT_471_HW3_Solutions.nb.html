# (a)
library(tidyverse)
library(zoo)

data("AirPassengers")

ap_df <- data.frame(
  Date = as.Date(as.yearmon(time(AirPassengers))),
  Passengers = as.numeric(AirPassengers)
)

ggplot(ap_df, aes(x = Date, y = Passengers)) +
  geom_line(color = "steelblue", linewidth = 1.2) +
  labs(
    title = "Monthly Airline Passenger Numbers (19491960)",
    x = "Year",
    y = "Number of Passengers (in thousands)"
  ) +
  theme_minimal(base_size = 14)

# (b)

ggplot(ap_df, aes(x = Date, y = Passengers)) +
  geom_line(color = "steelblue", linewidth = 1.2) +
  geom_smooth(color = "black", level = 0.90) +
  labs(
    title = "Monthly Airline Passenger Numbers (19491960)",
    x = "Year",
    y = "Number of Passengers (in thousands)"
  ) +
  theme_minimal(base_size = 14)

  1. We can observe that the number of passengers increase exponentially between the years 1950 and 1960. This means that more airline tickets are purchased as the number of passengers increase. If we had access to concrete data on ticket sales, we would be able to analyze more accurately how many tickets are purchased and whether they are affected by the increase in the number of passengers.
